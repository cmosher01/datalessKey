Contains implementations for the com.surveysampling.util.key package.

This package contains implementations of the {@link com.surveysampling.util.key.DatalessKeyAccess DatalessKeyAccess} and {@link com.surveysampling.util.key.DatalessKey DatalessKey} interfaces for each underlying datatype used to represent keys.

Developers of new datatypes can create a class with a name of the form DatalessKeyAccessDatatype, where Datatype is a name for the underlying datatype, which applications will specify as the datalessKeyUnderlyingTypeName argument in the {@link com.surveysampling.util.key.DatalessKeyAccessFactory#createDatalessKeyAccess DatalessKeyAccessFactory.createDatalessKeyAccess} method.
